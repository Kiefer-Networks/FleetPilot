FleetPilot — это мобильный клиент с открытым исходным кодом для Kandji, обеспечивающий полноценное управление MDM на телефоне, планшете и компьютере. Просматривайте весь парк устройств Apple, управляйте блюпринтами, работайте с автоматической регистрацией устройств, отслеживайте уязвимости и выполняйте действия с устройствами — всё в одном приложении с Material Design 3.

Возможности:
- Просмотр, поиск и фильтрация всех зарегистрированных устройств Mac, iPhone, iPad и Apple TV
- Детальный просмотр устройства с 8 вкладками: Обзор, Приложения, Статус, Активность, Команды, Секреты, Заметки, Местоположение
- Более 15 действий MDM: Блокировка, Перезагрузка, Выключение, Стирание, Режим пропажи, Удалённый рабочий стол и другие
- Управление блюпринтами: создание, редактирование, удаление, управление элементами библиотеки, загрузка OTA-профилей
- Автоматическая регистрация устройств (ADE): создание интеграций, загрузка токенов, просмотр зарегистрированных устройств
- Управление пользователями с курсорной пагинацией и поиском
- Управление уязвимостями: просмотр CVE по критичности, оценкам CVSS, затронутым устройствам
- Обнаружение угроз и поведенческие детекции с поиском
- Управление тегами с 10 цветовыми пресетами
- Журнал аудита с поиском по типу, администратору, устройству или пользователю
- Активность элементов библиотеки и статус развёртывания
- Обзор лицензий с лимитами устройств и разбивкой по платформам
- Мультитенантность: несколько профилей Kandji с мгновенным переключением
- Поддержка 25 языков
- Адаптивная компоновка: нижняя навигация на телефонах, навигационная рейка на планшетах и компьютерах

Безопасность:
- Обязательный 6-значный PIN-код при первой настройке
- Опциональная биометрическая разблокировка (Face ID, Touch ID, отпечаток пальца)
- Токены API хранятся в нативном защищённом хранилище платформы (iOS Keychain, Android Keystore)
- Для просмотра секретов устройства требуется биометрическая/PIN-верификация
- Деструктивные действия требуют текстового подтверждения
- Без отслеживания, без аналитики, без телеметрии

FleetPilot полностью бесплатен — без покупок в приложении, без подписок, без ограничений функциональности. Полностью открытый исходный код под лицензией MIT.
