FleetPilot은 Kandji를 위한 오픈소스 모바일 클라이언트로, 스마트폰, 태블릿, 데스크톱에서 완전한 MDM 관리를 제공합니다. Apple 기기 전체를 탐색하고, 블루프린트를 관리하고, 자동 기기 등록을 처리하고, 취약점을 추적하고, 기기 작업을 실행하세요 — Material Design 3 기반의 단일 앱에서 모두 가능합니다.

기능:
- 등록된 모든 Mac, iPhone, iPad, Apple TV 기기 탐색, 검색, 필터링
- 8탭 기기 상세 보기: 개요, 앱, 상태, 활동, 명령, 비밀, 메모, 위치
- 15개 이상의 MDM 기기 작업: 잠금, 재시작, 종료, 초기화, 분실 모드, 원격 데스크톱 등
- 블루프린트 관리: 생성, 편집, 삭제, 라이브러리 항목 관리, OTA 프로필 다운로드
- 자동 기기 등록(ADE): 통합 생성, 토큰 업로드, 등록된 기기 탐색
- 커서 기반 페이지네이션과 검색을 통한 사용자 관리
- 취약점 관리: 심각도, CVSS 점수, 영향받는 기기별 CVE 탐색
- 위협 탐지 및 행동 탐지(검색 기능 포함)
- 10가지 색상 프리셋을 포함한 태그 관리
- 유형, 관리자, 기기, 사용자별 검색이 가능한 감사 로그
- 라이브러리 항목 활동 및 배포 상태
- 기기 제한과 플랫폼 분류를 포함한 라이선스 개요
- 멀티테넌트 지원: 여러 Kandji 프로필 간 즉시 전환
- 25개 언어 지원
- 반응형 레이아웃: 스마트폰에서는 하단 내비게이션, 태블릿과 데스크톱에서는 내비게이션 레일

보안:
- 최초 설정 시 필수 6자리 PIN
- 선택적 생체 인증 잠금 해제(Face ID, Touch ID, 지문)
- API 토큰은 플랫폼 네이티브 보안 저장소에 저장(iOS Keychain, Android Keystore)
- 기기 비밀 확인 전 생체 인증/PIN 인증 필요
- 파괴적 작업 시 텍스트 입력 확인 필요
- 추적 없음, 분석 없음, 텔레메트리 없음

FleetPilot은 완전히 무료입니다. 인앱 구매, 구독, 기능 제한이 없습니다. MIT 라이선스에 따른 완전한 오픈소스입니다.
